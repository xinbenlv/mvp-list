# Family Getaway Planner — 5-Deck Image Prompts

Visual style for **all** decks (apply this as the base system prompt before each slide prompt):

> **Global style — Anthropic.ai aesthetic.** Warm off-white / cream background (#F4EFE6 ~ #EFE9DD), generous whitespace, editorial layout, premium calm feel. Typography: large serif headline (Tiempos / Söhne-style), small clean sans-serif for body. Accent color: muted coral / terracotta (#D97757) with secondary muted sage, dusty blue, and warm grey. Illustrations should feel hand-drawn / pencil-sketched, soft line weight, slightly imperfect, organic. No harsh gradients, no glossy 3D, no neon. Subtle paper grain. Centered composition with quiet confidence. 16:9 aspect ratio. No stock-photo people.

Slide text rules for **all** decks:

> Place the slide title in the **top-left corner** with generous margin, using a large warm serif headline. Keep slide body/subtitle text below the title in a small clean sans-serif. Preserve clear whitespace around all text so it remains readable and does not overlap the illustration.
>
> For Decks 2-5, include a quiet footer along the bottom edge in small warm-grey sans-serif text: **"MVP List, a Collaborated Weekend Planner powered by GBrain and GStack"** and **"https://b.zzn.im/blog/mvp-list/"**. Keep the footer subtle, aligned to the bottom-left or split left/right, and do not let it compete with the slide title.

## Notes for the image model

- If using **GPT Image / DALL·E**: paste the **Global style** paragraph and the **Slide text rules** first, then the per-deck prompt.
- For **Deck 3**, do a second pass if the model misspells "gbrain" — explicitly re-prompt: *"the center word must be the exact lowercase string g-b-r-a-i-n with no spaces, no hyphens, no other text inside the central circle."*

- For all slides, if the title or footer overlaps the illustration, regenerate with the instruction *"reserve a clear top-left title block and a clear bottom footer band; shift the illustration toward the center and lower-right."*
