Day Composer · Template System · v2

Three templates,
matched to your trip.

Same Day Composer engine, three visual personalities. Each template's color palette is derived from the trip's mood tags, not fixed — so a "warm reflective" day looks different from a "celebratory peak" day even within the same template.

The idea: persona × trip-nature → template

The 3 templates below cover the spectrum of how a day can feel: a keepsake, a magazine cover, a love poem. At intake time, the composer asks (or infers) what kind of day this is, and matches the right template — no "settings" page required.

The same trip rendered in 3 templates would still look different from another user's trip rendered in the same template — because the color palette is parameterized on that day's specific vibe tags + emotional arc shape + time-of-day. The template is the chassis; the day's energy is the paint.

How this becomes a product: At intake time, the composer reads the inferred mood tags (e.g. reflective + warm + family) and the emotional arc shape (slow-rise-then-fall), then picks the template + tunes the palette. The user can override ("I want it more 02"), but the default is always derived. The 3 chassis don't change; the paint does. ← This is the principle that makes the system scale from 3 templates to feeling like infinite personality.
